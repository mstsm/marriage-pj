# marriage-pj

## アプリの利用方法
1. スマートフォンのQRリーダーアプリを立ち上げてQRコードを読み込む。
1. ブラウザが立ち上がり、カメラ利用の許可が求められるので、カメラ利用を許可する。
1. カメラが起動されたら、ARマーカーにカメラを向ける。
